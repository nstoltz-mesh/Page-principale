# Page principale - Nicolas Stoltz Services Financiers

## Description
Site web principal pour les services de conseil financier de Nicolas Stoltz.

Conseiller en sécurité financière à Québec spécialisé dans :
- Gestion de placements
- Planification financière
- Assurances de personnes
- Coaching financier
- Déclaration de revenus

## Structure
- `index.html` : Page principale complète (HTML + CSS + JS intégrés)
- `images/` : Ressources visuelles (photo professionnelle)

## Design
- Palette : Vert #4e7a51, Teal #018476, Vert foncé #00221c
- Typographie : Playfair Display (titres) + Inter (texte)
- Style : Mix gradient moderne + cards sobres professionnelles

## Utilisation

## Déploiement
